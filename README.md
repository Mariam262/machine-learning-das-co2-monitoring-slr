# Distributed Acoustic Sensing for C02 Storage Monitoring - A systematic literature review

## Overview

This Systematic Literature Review (SLR) examines the use of Machine Learning (ML) and Deep Learning (DL) for real-time monitoring of subsurface CO₂ storage using Distributed Acoustic Sensing (DAS). DAS provides continuous, high-resolution fiber-optic monitoring data, making it a promising technology for tracking CO₂ plume movement, detecting leakage, identifying anomalies, and improving storage safety. However, DAS data are large, noisy, and complex, which creates a strong need for automated ML/DL-based interpretation.
This review analyzes recent studies on DAS-based monitoring, CO₂ storage surveillance, ML/DL models, signal-processing methods, performance metrics, and publication trends. After systematic screening and eligibility assessment, 9 studies were selected for detailed analysis. The findings show that DAS combined with ML/DL is still an emerging but promising research direction for intelligent, scalable, and real-time CO₂ storage monitoring.

## Research Highlights
Conducted a systematic literature review on DAS, ML/DL, and subsurface CO₂ storage monitoring.
Identified key DAS applications, including leakage detection, CO₂ plume monitoring, anomaly detection, and seismic monitoring.
Analyzed commonly used ML/DL techniques such as LSSVM, Random Forest, CNN, and Bayesian Optimization.
Reviewed performance metrics including accuracy, precision, recall, F1-score, RMSE, latency, detection reliability, and strain sensitivity.
Examined publication trends and research gaps in DAS-based real-time CO₂ storage monitoring.
